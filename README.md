## 核心功能
- 视频采集 ：支持摄像头设备管理和视频采集

- 媒体处理链 ：实现了基于链式结构的媒体处理流程

- 视频渲染 :视频预览,渲染窗口管理及视频帧显示

- 网络传输 ：基于 ICE 协议实现网络连接和数据传输
  
- 会话管理 ：通过 SDP Offer/Answer 模型建立对等连接
  
- RTP/RTCP 协议 ：实现了 RTP 打包和 RTCP 反馈机制
  
- 视频推流 : 推流接口封装及视频编码帧发送
